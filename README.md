# Matemática Interativa

Projeto de desenvolvimento de materiais matemáticos interativos utilizando SageMath, Python e LaTeX.

## Sobre o projeto

Este projeto tem como objetivo criar demonstrações e visualizações interativas de conceitos matemáticos, combinando explicação teórica, linguagem matemática e recursos computacionais.

## Objetivos

- Explorar construções geométricas interativas;
- Relacionar definições matemáticas com visualizações;
- Criar materiais de apoio para o ensino de matemática.

## Tecnologias

- SageMath
- Python
- LaTeX
- GitHub
